HOW TO USE GIT IN POWERSHELL !

How to clone your repository

  • "cd" changes directory.
  • Choose directory.
  • "Git clone" + ssh key

    • Upload things to your git!

  • "Git status"
  • "Git add *"
  • "Git commit -m " " "
  • "Git push"

    • Branches

    "Git branch": Shows you which branch you're in

    "Git branch name": Adds a new branch

    "Git checkout": To change your branch

    "Git merge": Merges your branches

    Merge branches

  • Change to prefered branch
  • Git add, commit and push.
  • Change branch
  • Use same commands.
  • Go to your prefered branch. (Main is the best choice)
  • Git merge "other branch"
  • Check on git
